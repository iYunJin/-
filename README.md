# 现代控制理论

## 控制系统的状态空间表达式

### 状态变量及状态空间表达式    
* 状态变量
1. 足以完全确定系运动状态
2. 个数又是最少
* 状态矢量  
n个状态变量用x1(t),x2(t),...,x(t)表示，并把这些状态变量看做是矢量***x***(t)的分量，则***x***(t)就称为状态矢量
![image](https://github.com/user-attachments/assets/c63e48a4-aaad-42ff-aa9c-55ce934b5063 "状态矢量")

* 状态空间  
以状态变量为坐标轴所构成的n维空间
* 状态方程  
由系统的状态变量构成的一节微分方程组称为系统的状态方程。
设单输入单输出定常系统，矢量矩阵表示时表达式为：
```
\$\dot x$ = Ax + bu
y = cx
```
